PATh User Portal

Welcome to the PATh Facility User Portal, a website designed to centralize resources Path Facility Users might need.

The PATh Facility is a purpose-built, national-scale distributed High Throughput Computing (dHTC) resource meant to deliver computational capacity to NSF researchers. Like other HTC resources, the PATh facility specializes in running 'scaled out' computational workloads, running thousands of independent jobs concurrently.

If you are interested in using PATh Facility resources application information can be found in the NSF Dear Colleague Letter (NSF 22-051)
